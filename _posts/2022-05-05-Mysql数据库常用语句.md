## 删除表中数据

- TRUNCATE TABLE
  - 删除表中的所有行，而不记录单个行删除操作。
  - TRUNCATE TABLE name
  
- 区别

  - TRUNCATE TABLE 在功能上与不带 WHERE 子句的 DELETE 语句相同：二者均删除表中的全部行。但 TRUNCATE TABLE 比 DELETE 速度快，且使用的系统和事务日志资源少。

- TRUNCATE TABLE 清空表所有数据，而且使标识列从标识种子(一般标识种子从1开始)重新开始
